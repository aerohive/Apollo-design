# Layout 布局 提供页面的整体布局。 ## 使用方式 ### 样式说明 | class | 说明 | | ------------- | ------------- | | clearfix | 用来清除浮动| | grid_n | n从1到20,以62px递增至1220px| | column | 使元素左浮动且定义了与左右相邻元素的margin值(10px)| | first | 使元素左margin为0| | last | 使元素右margin为0 |
