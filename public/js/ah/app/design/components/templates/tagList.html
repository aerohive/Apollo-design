# TagList 标签 标签 ## API ### Property | 参数 | 说明 | 类型 | 可选值 | 默认值(Default) | | -------------| ------------- | ----- | ------ | --------------- | | target | 取得tag列表的URL | String | URL | '' | | items | 自定义标签列表内容 | Array | [{}, {}] | | | ### Methods | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | refresh | 刷新标签列表 | function( d : Array )| ### Events | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | mouseEnter | 当鼠标移到一个tag时，触发此事件 | function( type:String) | | mouseLeave | 当鼠标离开一个tag时，触发此事件 | function( type:String) | | itemClose | 关闭一个tag时，触发此事件 | function( type:String) | | itemRevert | 恢复一个tag时，触发此事件 | function( type:String) |
