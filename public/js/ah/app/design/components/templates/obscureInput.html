# ObscureInput隐藏输入 隐藏输入 ## API ### Property | 参数 | 说明 | 类型 | 可选值 | 默认值(Default) | | ------------- | ------------- | ----- | ------ | --------------- | | tabTitles | 用来生成几个tab页 | String | 'tab1,tab2' | N/A | | cur | 初始化第几个tab处于current状态 | Number | 0,1,2,3,4... | 0 | | theme | 主题样式 | String | gray | N/A | | postion | tab水平/垂直位置 | String | vertical | horizontal | ### Methods | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | show | 动态显示哪个tab | function( i : Number ) | | hide | 动态隐藏哪个tab | function( i : Number ) | ### Events | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | tabClick | 当点击各个tab项的时候，触发此事件 | function( tab:Element, content:Element, i:Number ) |
