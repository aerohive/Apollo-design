# Menu 弹出菜单 弹出菜单 ## API ### Property | 参数 | 说明 | 类型 | 可选值 | 默认值(Default) | | ------------- | ------------- | ----- | ------ | --------------- | | items | 用来生成弹出菜单项目 | Object或Array | 自定义,详见Demo | N/A | | btnText | 按钮的标题 | String | String | ‘Test’ | | theme | 主题样式 | String | medium,large | N/A | | action | children 触发事件类型 | String | mouse,click | mouse | | type | 设置需要显示的菜单集 | String | 自定义,仅当itmes是Object可用 | __ | ### Methods | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | show | 显示哪些菜单项 | function( list : Array ) | | hide | 隐藏哪些菜单项 | function( list : Array ) | ### Events | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | hook | 当点击某个菜单项且其无子菜单时触发此事件 | function( String:type,e ) | | type | 当点击某个菜单项且其无子菜单时触发此事件 | function() | Events中的type，从items菜单项中获取，如果没有配置type，组件通过处理name获得。
