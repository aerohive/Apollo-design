# PasswordLevel密码强度提示框 密码强度提示框 ## API ### Property | 参数 | 说明 | 类型 | 可选值 | 默认值(Default) | | -------------| ------------- ------------- | ----- | ------ | --------------- | | bLen | 密码长度大于8 | Boolean | true/false | true | | bNumber | 密码中必须包含数字 | Boolean | true/false | true | | rType | 密码中必须包含的自定义字符类型 | String | "special"/"upper" | "special" | ### Events | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | focus | 输入框获得焦点时触发 | function( e:event )| | keyup | 输入框当按钮被松开时触发 | function( e:event )| | blur | 输入框失去焦点时触发 | function( e:event )|
