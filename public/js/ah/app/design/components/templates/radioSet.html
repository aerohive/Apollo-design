# RadioSet 单选 这是一种特殊样式的单选,在我们的项目很多地方用到 ## API ### Property | 参数 | 说明 | 类型 | 可选值 | 默认值(Default) | 注意| | ------------- | ------------- | ----- | ------ | --------------- | | items | 必选项 | Array | {name:'1 Hr',type:'rangeOption',value:1} | '' |type为自定义事件名称 | | cur | 可选项 | int | 0,1,2,3... | 0 |设置默认选择项 | ### Methods | 事件 | 说明 | 类型 | | ------------- | ------------- | ----- | | show | 手动设置当前选中第几个 | show : function(i) {} | | hide | 隐藏部分选项 | function( "1,2,3" ) | | rangeOption(自定义事件,传入的时候定义的) |当选项改变时触发的操作,供使用者调用自己的逻辑 | |
