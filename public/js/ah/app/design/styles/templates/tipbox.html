# Tipbox 提示框 提供error, warning, success 多种形式的提示框样式。 ## 使用方式 ### 支持的样式列表 | class | 说明 | | ------------- | ------------- | | ui-tipbox | 基础类名, 提供标准尺寸的提示框| | ui-tipbox-large, ui-tipbox-small | 定义icon比较大和比较小的提示框 | | ui-tipbox-error, ui-tipbox-warning, ui-tipbox-success | 定义提示框类型 | | ui-tipbox-plain | 定义空白背景的提示框| ### class叠加顺序 基础类：ui-tipbox | ui-tipbox-small | ui-tipbox-large &darr; 定义类型：ui-tipbox-error | ui-tipbox-warning | ui-tipbox-success &darr; 定义plain ：ui-tipbox-plain
