# Upload 上传组件 实现客户端单文件/多文件上传功能，继承自dojox/uploader。 ## API ### Property 继承自dojox/form/uploader | 参数 | 说明| 类型 | 可选值| 默认值(Default) | | ------------- | ------------- | ----- | ------| --------------- | | uploadOnSelect| 是否在选择文件后立即上传| Boolean | true/false | false | | multiple | 是否支持多文件上传 | Boolean | true/false | false| | url | 文件上传地址| String | '' | '' | | name | 提交到后台的字段名 | String | ''| uploadedfile | | flashFieldName| 提交到后台的flash字段名 | String | ''| Flash | | force | 上传方式 | String | form/html5/iframe/flash | html5/iframe | | showInput | 显示选中的文件名的input的位置 | String | before/after|''| | focusedClass | 按钮focus状态时添加的class名称 | String | ''| dijitButtonHover| | baseClass | 按钮的基础样式class | String | ''| dijitUploader dijitButton | 其他参数 | 参数 | 说明| 类型 | 可选值| 默认值(Default) | | ------------- | ------------- | ----- | ------| --------------- | | extraParams | 上传url地址需要附加的参数 | String | ''| null | | allowExtension | 允许上传的文件扩展名 | Array | jpg/jpeg/png/gif | [] | | sizeLimit | 允许多文件上传时的文件个数限制 | Number | | 0 (无限制) | ### Methods | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | upload | 执行文件上传动作，实际在各种plugin中支持 | function( formData : Object ) | | submit | 当上传组件在一个form表单中，且表单中的其他字段需要一起提交时，此方法用于替代form默认的submit | function( formNode : Node ) | | reset | 重置上传组件，清除所有已选中文件 | function() | | msg | 需要消息提示时调用的函数 | function (msg : String) | ### Events | 参数 | 触发条件 | 类型 | | ------------- | ------------- | ----- | | change | 选中的文件发生变化 | function( fileArray: Array ) | | begin | 开始上传文件 | function( dataArray: Array ) | | progress | 上传过程中，且上传类型不是iframe | function( customEvent: { bytesLoaded: 已上传字节数, bytesTotal: 总字节数, type: 事件类型(progress/load), timeStamp: 开始上传时间} ) | | complete | 上传完成 | function( customEvent: Object ) | | cancel | 未选择文件时关闭对话框 | function( ) | | abort | 取消上传 | function( ) | | error | 上传出错 | function (evtObject : Object) |
