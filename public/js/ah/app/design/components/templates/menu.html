# Menu菜单栏 菜单栏 ## API ### Property | 参数 | 说明 | 类型 | 可选值 | 默认值(Default) | | ----------| ------------- | ----- | ------ | --------------- | | bReplace | 是否出现目录标题 | Boolean | true,false | true | | btnText | 设定目录的标题内容 | String | 'Assign'... | 'Test' | | wrapEl | 设定目录栏的父级标签 | String |'&lt;div&gt;&lt;/div&gt;'...|'&lt;div&gt;&lt;/div&gt;'| | theme | 设定目录栏的宽度 | String | 'medium'/'large' | N/A | | items | 添加目录栏中的子项 | Array | [{},{},{}...] | N/A | ### Methods | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | getChildrenData | 获取某一栏中的所有子项的数据 | function( type : String )| | show | 设定某一栏显示 | function( list : String,Array )| | hide | 设定某一栏不显示 | function( list : String,Array )| ### Events | 参数 | 说明 | 类型 | | ------- | ------------- | ----- | | name(某一栏的name) | 当点击某一栏时，触发此事件 | function( type:Element) | | hook | 当点击菜单栏中任意栏时，均触发此事件 | function( type:Element) |
