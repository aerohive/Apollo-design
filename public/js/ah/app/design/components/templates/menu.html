# Menu选择框 选择框 ## API ### Property | 参数 | 说明 | 类型 | 可选值 | 默认值(Default) | | ---- | ---------| ----- | ------ | --------------- | | items | 渲染所需的数据 | Array/Object | [{},{}]/{} | N/A | | bReplace | 是否对原dom节点进行替换 | Boolean | true,false | true | | btnText | 选择框显示的文字,在bReplace为true时有效 | String | any | 'Test' | | action | 设置二级节点显示的动作是靠鼠标移入还是点击 | String | 'mouse','click' | 'mouse' | | theme | 主题样式，设置选择框样式和大小 | String | 'medium','large' | N/A | ## Method ### Property | 参数 | 说明 | 类型 | | ---- | ----- | ---- | | show | 显示某条或多条记录 | function(list: []) | | hide | 隐藏某条或多条记录 | function(list: []) | ## Events | 参数 | 说明 | 类型 | | ---- | ---- | ---- | | hook | 点击某条选项时触发的事件 | function(type) |
