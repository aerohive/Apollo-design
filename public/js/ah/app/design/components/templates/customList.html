# CustomList列表 列表 ## API ### Property | 参数 | 说明 | 类型 | 可选值 | 默认值(Default) | | -------------| ------------- | ----- | ------ | --------------- | | scrollHeight | 列表高度 | Number | 大于56 | 56 | | scrollLimit | 设定列表中的滚动条 | Number | 1,2,3... | 12 | | noItemMsg | 设定列表内容为空时的提示信息 | String | any | 'No Records found' | | listType | 列表类型 | String | 'osDhcp'/'osHttp' | N/A | | item | 列表的内容 | Array | ['','',''...] | N/A | ### Methods | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | isPredefined | 判断输入框中的值与列表中的内容是否相同 | function( v: String ) | ## Events | 参数 | 说明 | 类型 | | ---- | ---- | ---- | | ItemClick | 当点击列表中某一栏时，触发此事件 | function(type: String) | | blur | 当输入框失去焦点时，触发此事件 | function() |

Demos

<input data-dojo-type="ah/util/layout/list/CustomList" data-dojo-props=" scrollLimit:2, scrollHeight: 70, items: ['Android', 'Apple iOS', 'Blackberry', 'CrOS']">
