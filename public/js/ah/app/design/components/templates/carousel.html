# Carousel轮播 图片轮播 ## API ### Property | 参数 | 说明 | 类型 | 可选值 | 默认值(Default) | | ------------- | ------------- | ----- | ------ | --------------- | | showNum | 一次显示的图片个数，只有当总的图片个数大于showNum时，滚动箭头才会显示 | Number | 1,2,3,4... | 1 | ### Methods | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | refresh | 动态显示轮播的内容 | function( i : NodeList ) |
