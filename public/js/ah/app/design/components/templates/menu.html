# Menu菜单栏 菜单栏 ## API ### Property | 参数 | 说明 | 类型 | 可选值 | 默认值(Default) | | ---- | ---------| ----- | ------ | --------------- | | items | 渲染所需的数据 | Array/Object | [{},{}]/{} | N/A | | bReplace | 是否对原dom节点进行替换 | Boolean | true,false | true | | btnText | 菜单栏显示的文字,在bReplace为true时有效 | String | any | 'Test' | | action | 设置二级菜单显示的动作是靠鼠标移入还是点击 | String | 'mouse','click' | 'mouse' | | theme | 主题样式，设置菜单栏样式和大小 | String | 'medium','large' | N/A | ## Method ### Property | 参数 | 说明 | 类型 | | ---- | ----- | ---- | | show | 显示某条或多条记录 | function(list: []) | | hide | 隐藏某条或多条记录 | function(list: []) | ## Events | 参数 | 说明 | 类型 | | ---- | ---- | ---- | | hook | 点击某条菜单时触发的事件 | function(type) | | type | 点击定义此type菜单时触发的事件，type为items中配置的type | function() |

Demos

// template.html <div data-dojo-stype="ah/util/layout/Menu" data-dojo-props="items: [{name: 'item1'},{name: 'item2'}], btnText: 'item'"></div> // template.js events: [ ['menuItem', 'item1', '_handleHook'] //只在点击item1时触发事件 ], _handleHook: function() { //do something }
