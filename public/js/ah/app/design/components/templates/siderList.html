# SideList列表 列表 ## API ### Property | 参数 | 说明 | 类型 | 可选值 | 默认值(Default) | | -------------| ------------- | ----- | ------ | --------------- | | bExpand | 设定列表是否默认展开 | Boolean | true,false | true | | bDot | 设定是否带列表提示图片 | Boolean | true,false | true | | items | 添加列表中的内容 | Array | [{},{},{}...] | N/A | | hideArea | 隐藏列表中的某几栏 | String/Array | ''/['',''...] | N/A | ### Methods | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | hide | 动态隐藏列表中的某几栏 | function( d : String/Array )| | show | 动态显示列表中的某几栏 | function( d : String/Array )| ### Events | 参数 | 说明 | 类型 | | ------------- | ------------- | ----- | | itemClick | 当点击列表中某一栏时，触发此事件 | function( type:String) |
